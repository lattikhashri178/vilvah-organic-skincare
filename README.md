# 🌿 Vilvah – DBMS Project

## 📌 Project Overview

This project focuses on designing a Database Management System (DBMS) for **Vilvah**. The objective is to analyze the business requirements and design a structured relational database that efficiently manages customer information, products, inventory, suppliers, orders, payments, and administrative operations. The project is developed in multiple phases, including system analysis, database design, and implementation.
