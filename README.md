Vilvah
Vilvah is a Database Management System (DBMS) project based on an Indian natural skincare and personal care brand. This project focuses on designing and managing the database required to handle the company's day-to-day business operations efficiently.
The system stores and manages essential business information such as products, customers, suppliers, inventory, orders, and payments. It demonstrates how a well-designed database helps organize data, reduce redundancy, improve accuracy, and support business processes.
